# Themes

Hướng dẫn làm themes cho app.

Sau khi cài đặt app xong thì thêm sẽ có cấu trúc như sau:

```
--apps
	|---modules
			|---Name Module
					|---views
	|---views
	        |---ajax
	        |---comment
	        |---elements
	        |---friend
	        |---home
	        |---include
	        |---layouts
	        |---profile
	        |---user
	        |---webroot

```
| Path | Description |
| -- | -- |
| app/views | Chứa các giao diện chính của app |
| app/views/webroot | Chứa các thu viện css, js, images |
